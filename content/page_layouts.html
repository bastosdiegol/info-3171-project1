Page Layout

Our responsive page layout for the dashboard is designed to adapt to various screen sizes. The layout includes a sidebar for single-value charts and 6 chart blocks to display multi-value charts.

Layout Configurations

Standard Layout

Structure

  • Sidebar: Occupies the left side.
  • Chart Blocks: Displayed in a two-column format on the right side.

Mobile Layout

Structure

  • Sidebar: Hidden to maximize space for chart blocks.
  • Chart Blocks: All chart blocks stack vertically, with an additional chart block replacing the sidebar.

Grid Gutter

Our grid columns have a 15px padding on either side, which results in a 30px gutter between columns and a 15px gutter on the grid edges. Even though our columns are fluid and responsive, the gutter remains constant, ensuring consistent spacing across all screen sizes.

Responsiveness

Our responsive page layout automatically change configuration structure once it reaches certain breakpoints.
